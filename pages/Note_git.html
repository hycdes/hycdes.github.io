git开始使用和基本方法

(1) 本机首次使用git需要设定上自己的github帐号：

git config --global user.name 'xxxxx'

git config --global user.email 'xxx@xx.xxx'

(2) 进入目录，例如E盘hycdes：

cd e:/hycdes

(3) 设为仓库，随后文件夹会生成隐藏文件.git：

git init

(4) 生成本机SSH Key，并复制添加到github-setting-SSH and GPG keys里：

ssh-keygen -t rsa -C 'xxx@xx.xxx'

*注意：该命令将在当前目录生成privateKey和publicKey文件，请务必放在默认目录下(Windows即C:\Users\xxxx\.ssh\)

(5) 添加远程仓库地址，进入github找到仓库，绿色按钮clone or download，复制里面的地址xxxx：

git remote add origin xxxx

*注意：github.io个人主页绑定其他域名后仓库会生成CLIKE文件，需要再次pull，否则push会报错

(6) 在文件夹里添加文件：

git pull origin master

(7) 添加文件：

git add .

(8) 提交信息：

git commit -m 'xxxxxx'

(9) 同步上去，-u第一次用，后来可以省去：

git push -u origin master

 

(ex) 复制clone地址，下载到本地：

cd e:/download

git clone xxxx

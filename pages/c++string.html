string 方法

基本方法

#include<string>

string str; char c; char* C;

 

getline(cin,str);  //输入可带空格的一行字符串

str.length();  //返回str长度

str += c;  //直接在str后插入c

str.substr(int p, int n);  //提取str从p往后n长度子串

string &replace(int p, int n, const char* C);  //从p开始的n个字符替换成C[]

string &replace(int p, int n, string &str);  //从p开始的n个字符替换成str

string &replace(int p, int n, string &str, int pos, int num);  //从p开始的n个字符，替换成从str的pos开始的num长度子串

数字和string互相转换

需要 #include<sstream>，整数浮点皆可

字符串转数字（整形为例）：

string str; sstream ss; int n;

ss << str;

ss >> n;

数字转字符串：

ss << n;

str = ss.str();
